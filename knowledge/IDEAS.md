### Пример кода (Python с использованием PyGithub):
```python
from github import Github

# Авторизация через токен
g = Github("YOUR_GITHUB_TOKEN")

# Получение репозитория
repo = g.get_repo("username/repository")

# Чтение файла
file = repo.get_contents("path/to/file.md")
print(file.decoded_content.decode())

# Запись изменений
repo.update_file(
    "path/to/file.md",
    "Update file via script",
    "Новое содержимое файла",
    file.sha
)
```
