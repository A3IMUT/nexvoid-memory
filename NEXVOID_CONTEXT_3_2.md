## 🔧 ИЗВЕСТНЫЕ ОСОБЕННОСТИ И РЕШЕНИЯ

- Ghost Settings API (PUT /settings/) не поддерживает ключ `navigation` в версии 5.130 — обходим через прямой UPDATE в MariaDB
- MariaDB доступна через: `docker exec ai-media-mariadb mariadb -u root -pSuperSecureRoot2026! ghost`
- `key` — зарезервированное слово в MariaDB, в SQL-запросах оборачивать в обратные кавычки: `` `key` ``
- Ghost Admin API требует JWT (HS256) с полем `kid` в header и `aud: "/admin/"` в payload

### Memory Curator — правила

Главный принцип:

Точность > Креативность

Memory Curator сохраняет знания максимально близко к исходному сообщению пользователя.

Запрещено:

- придумывать факты;
- придумывать версии API;
- придумывать технологии;
- придумывать инструменты;
- придумывать метрики;
- делать предположения о намерениях пользователя;
- расширять идею без явного указания пользователя.

При недостатке данных:

- использовать исходный текст;
- сохранять краткое описание;
- не выдумывать детали.

### UX-правило

После успешного сохранения должно отправляться только одно подтверждение.

Не использовать одновременно:

1. Идея сохранена в памяти
2. Идея успешно сохранена в GitHub

Оставлять только одно итоговое сообщение.
